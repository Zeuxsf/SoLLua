# Previsão do Sol e da Lua

### Oque esse código faz?
Ele usa uma API da MeteoBlue para receber informações sobre o tempo, clima e etc. O código vai mostrar somente a previsão sobre o pôr e o nascer dos astros (Sol e Lua), e sempre o do dia seguinte ao atual.

### Porque esse código foi criado?
Esse código foi criado para por em prática uma habilidade simples, que é a de conseguir pegar as informações de uma API e usar elas no seu próprio código.

### Como eu posso utilizar esse código?
Simples:
- Baixe o repositório na sua máquina.
- Crie um ambiente virtual.
- Instale os pacotes: `pip install -r requirements.txt`
- Crie um .ENV
- Crie uma conta na [MeteoBlue](https://www.meteoblue.com)
- Pegue sua KEY e cole no .ENV. Ex.: `KEY=keydameteoblue123`
